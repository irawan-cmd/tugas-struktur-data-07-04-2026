# tugas-struktur-data-07-04-2026
visualisasi animasi pencarian jalur keluar dari sebuah labirin menggunakan
algoritma **(BFS)** Bredth-first search di buat dengan python.

Fitur yang ada:

Menu interaktif — pilih BFS, DFS, A*, atau bandingkan semua
Warna di terminal — merah=dinding, hijau=jalur, kuning=explored, cyan=posisi current
Hasil detail — panjang jalur, sel dieksplorasi, waktu, urutan langkah (→ ↓ ← ↑)
Tabel perbandingan — langsung lihat perbedaan ketiga algoritma
